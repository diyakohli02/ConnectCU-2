# Student Connect Portal (Frontend)

Student Connect Portal is a web-based networking platform designed for students of Chitkara University to connect, collaborate, and discover peers with similar skills and interests. The platform works as a **mini LinkedIn for students**, allowing them to build profiles, explore communities, and interact with fellow students.

This repository contains the **frontend implementation** of the project developed using modern web technologies.

---

## Project Overview

The goal of the Student Connect Portal is to provide a centralized platform where students can:

- Create and showcase their profiles
- Highlight their skills and interests
- Discover and connect with peers
- Join communities based on shared interests
- Collaborate on academic projects and events

The application focuses on providing a **responsive and interactive user interface** to enhance the overall user experience.

---

## Tech Stack

The frontend of the project was built using the following technologies:

- **React.js** – For building a modular and dynamic user interface
- **JavaScript** – For implementing application logic and interactivity
- **CSS** – For styling and responsive layouts
- **React Router** – For seamless navigation between different pages
- **Visual Studio Code** – Development environment used for building the project

---

## Features

### Dynamic Student Profiles
Students can create profiles where they can showcase their background, skills, and interests.

### Communities and Interest Groups
Users can explore communities based on topics such as coding, sports, design, or other academic interests.

### Peer-to-Peer Connections
Students can connect with other students who share similar skills or goals.

### Responsive User Interface
The application is designed to work smoothly across different screen sizes including desktops, tablets, and mobile devices.

### Modular React Components
The portal is built using reusable React components which improves maintainability and scalability.

## Installation and Setup

Follow these steps to run the project locally:

1. Clone the repository: git clone https://github.com/ChitraKalra17/Student-Connect-Portal.git
2. Navigate to the project folder: cd student-connect-portal
3. Install dependencies: npm install
4. Start the development server: npm start
The application will run on: http://localhost:3000
